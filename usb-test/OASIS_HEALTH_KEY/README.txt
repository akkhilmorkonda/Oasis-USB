HOW TO OPEN THE PATIENT RECORD
------------------------------
1. Open the file: index.html
2. Double-click index.html (or right-click -> Open with -> your web browser)
3. The patient record will load automatically. No internet required.

WHAT'S INCLUDED
---------------
- index.html     : Main offline viewer
- styles.css     : Viewer styling
- app.js         : FHIR parsing and rendering
- fhir/bundle.json : Single-patient FHIR Bundle
- media/         : Images, audio, video files referenced in the record

TECHNICAL NOTES
---------------
- Data is stored in FHIR (Fast Healthcare Interoperability Resources) format
- This is a read-only viewer. No edits are saved.
- Works offline. No installation required.
- Compatible with any modern web browser.

For more information about Oasis, visit the project repository.
